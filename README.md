# open-event-ios
**iOS** app for **Open Event**

## Introduction
> **This is an iOS app developed for `FOSSASIA` in mind.**

## Things To Have
1. **Xcode**
2. **[CocoaPods](http://cocoapods.org/)**

## Setting Up
1. Clone this repository
2. `cd` to this repository
3. Run `pod install`
4. Open `.xcworkspace` file *(This is **important**)*
